# Real-Time Chat Application

## Features
- One-to-one chat
- Group chat
- Typing indicator
- Read receipts
- Online/offline status
- Profile picture upload
- Redis pub/sub
- MongoDB integration
- Socket.IO with NestJS

---

## Tech Stack
- NestJS
- MongoDB
- Redis
- Socket.IO

---

## Setup Instructions

### 1. Clone repo
git clone <repo-url>

### 2. Install dependencies
npm install

### 3. Create .env file
Copy from .env.example

### 4. Start MongoDB
mongod

### 5. Start Redis
redis-server --port 6379

### 6. Run project
npm run start:dev
